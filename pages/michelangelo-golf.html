애드캡슐소프트

2023.12 — 2024.01 Java17 Spring Boot MariaDB Docker JPA Swagger

프로젝트 개요

골프장 소개 및 리뷰 기능을 제공하는 콘텐츠 기반 웹 서비스 구축 프로젝트에 참여하였습니다.
관리자에서 등록한 콘텐츠를 사용자에게 제공하고, 리뷰 등록 및 관리 기능을 포함한 서비스입니다.


본 프로젝트는 소규모 인원으로 단기간에 진행되었으며,
저는 백엔드 개발자로 참여하여 리뷰 기능 및 관리자 콘텐츠 조회 API 구현과 공통 처리 구조 적용을 담당하였습니다.


또한, 프로젝트 진행 과정에서 JWT 기반 인증 및 Spring Security 설정, Swagger를 활용한 API 문서화, Docker 및 호스팅 환경 구성 등을 경험하며 서비스 운영에 필요한 전반적인 기술 스택에 대한 이해도를 높일 수 있었습니다.

주요 업무

  • REST API 개발
  • JPA Specification 동적 조회 기능 구현
  • Docker 배포 환경 설정 및 구조 학습
  • JWT 인증 구조 학습 및 적용

기술 스택

Java17 Spring Boot MariaDB Docker JPA Swagger

회고

해당 프로젝트를 통해 인증, 문서화, 배포 환경 등 서비스 운영에 필요한 다양한 기술 요소들을 접하며 백엔드 개발에 대한 이해 범위를 확장할 수 있었습니다.

특히, Spring Security 기반 인증 구조와 Swagger를 활용한 API 문서화, Docker 및 호스팅 환경을 간접적으로 경험하며 단순 기능 구현을 넘어 서비스가 운영되는 전반적인 흐름을 이해하는 계기가 되었습니다.

다만, 기존 프로젝트는 Java 8 환경을 사용해왔던 것과 달리 본 프로젝트에서는 Java 17 환경이 적용되어 있었음에도 불구하고, record와 같은 최신 문법 및 기능을 실제 코드에 적극적으로 활용해보지 못한 점은 아쉬움으로 남았습니다.

이를 통해 새로운 기술이나 언어 기능을 단순히 이해하는 것에 그치지 않고, 실제 프로젝트에 적용하며 체득하는 것이 중요하다는 점을 느끼게 되었으며, 이후에는 최신 기술을 적극적으로 활용해보려는 방향으로 학습하고 있습니다.
